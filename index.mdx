	hesis-java-framework нь дипломын ажлын хүрээнд хөгжүүлсэн **Annotation-д суурилсан Java IoC/DI** фреймворк юм. 
Spring болон Guice-ийн үндсэн зарчмыг дагасан боловч **рефлекшн кэшлэлт (reflection caching), шууд болон хүлээгдэх (eager/lazy) инстанс үүсгэлт, байгуулагчийн хамаарал оруулалт (constructor injection), амьдралын мөчлөгийн удирдлага (lifecycle), тогтвортой класс уншилт (deterministic scanning), гүйцэтгэлийн хэмжилтийн модуль** зэргийг багтаасан. 
Мөн төслийн хүрээнд **хөгжүүлэгчийн тав тух (DX - Developer Experience)**-ыг дээшлүүлэхэд түлхүү анхаарсан болно.
