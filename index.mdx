`thesis-java-framework` нь дипломын ажлын зорилгоор хөгжүүлсэн, **Annotation-д суурилсан Java IoC/DI** фреймворк юм.
Тус санг хөгжүүлэхдээ Spring болон Guice зэрэг системийн үндсэн зарчмуудад тулгуурласан бөгөөд дотоод бүтэцдээ **рефлекшн кэшлэх (reflection caching), класс инстансыг шууд болон хүлээлттэйгээр үүсгэх (eager/lazy initialization), байгуулагчийн хамаарал оруулалт (constructor injection), амьдралын мөчлөгийн удирдлага (lifecycle), тогтвортой дараалалтай уншилт (deterministic scanning) болон гүйцэтгэлийн хэмжилтийн (benchmark) дэд модуль** зэрэг орчин үеийн боломжуудыг багтаасан.
Техникийн шийдлээс гадна **хөгжүүлэгчийн тав тухыг (DX - Developer Experience)** дээшлүүлэхэд онцгой анхаарал хандуулсан болно.
